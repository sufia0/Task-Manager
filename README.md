**TaskFlow** is a full-stack project management application designed to streamline team collaboration. It features secure authentication, interactive task boards, and a seamless responsive design. Built with the MERN stack (PostgreSQL version) and deployed for high availability.

## ‚ú® Key Features

### üîê **Authentication & Security**
- Secure User Registration & Login.
- JWT-based authentication with HTTP-only cookies.
- Password encryption using BCrypt.

### üìã **Task Management**
- **Create, Read, Update, Delete (CRUD):** Full control over tasks.
- **Board View:** Visualize workflow stages (To-Do, In Progress, Completed).
- **Task Details:** Add descriptions, priorities, and deadlines.

### üé® **Modern UI/UX**
- **Responsive Design:** Fully optimized for mobile and desktop using **Tailwind CSS**.
- **Interactive Modals:** Smooth transitions for adding/editing tasks.
- **Toast Notifications:** Real-time feedback for user actions.
